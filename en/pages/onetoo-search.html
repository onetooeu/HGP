ONETOO Search (EN)

ONETOO Search is a deterministic “AI index” node. This page gives a practical guide for humans and agents: how to query, interpret results, and build an allowlist workflow.

UI & API

  • UI: https://search.onetoo.eu/
  • API example: /search/v1?q=hgp&limit=10

Agent query discipline

  • Always set limit (10/25) for auditability.
  • Use cache-busting when auditing: ?cb=TIMESTAMP.
  • Log query intent + allowlist context.

Safety

  • Treat search results as pointers, not truth.
  • Trust comes only after manifest verification (see Trust & Verify).
HGPeDU is a static trust node: integrity via .well-known/sha256.json + minisign.
If you changed anything, regenerate and re-sign the manifest before deployment.
