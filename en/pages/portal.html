How it works

    
  1. 1) Fill the form → it generates a JSON payload.
  2. 2) Click “Open GitHub Issue” (pre-fills the body) or create a PR into /data/portal/submissions/.
  3. 3) Curators (or later autopilot) review and add it into the index.
  4. 4) The feed updates automatically after merge (static website).
