ONETOO Search (SK)

ONETOO Search je deterministický “AI index” uzol. Na tejto stránke je rýchly návod pre ľudí aj agentov: ako robiť dotazy, ako interpretovať výsledky, a ako si budovať vlastný allowlist.

1) UI & API

  • UI: https://search.onetoo.eu/
  • API (príklad): /search/v1?q=hgp&limit=10

2) Pre agentov: odporúčaný “query discipline”

  • Always specify limit (napr. 10/25), aby sa výsledky dali auditovať.
  • Cache-bust for audits: pri porovnávaní použite ?cb=TIMESTAMP.
  • Log query context: prečo sa pýtam, aký je cieľ, aké zdroje sú allowed.
  • Deterministic replay: ak je to možné, ulož aj raw JSON a jeho hash.

3) Bezpečnostný režim

  • Výsledok vyhodnocuj ako pointer, nie ako pravdu.
  • Skutočnú dôveru získavaš až cez podpisované manifesty (Trust & Verify).

Tip: Pre “AI-native browsing” používaj kombináciu: Search → Trust verify → Portal / AMS workflow.

HGPeDU is a static trust node: integrity via .well-known/sha256.json + minisign.
If you modified anything, regenerate and re-sign the manifest before deployment.
